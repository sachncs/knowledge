## Summary

Indent your code blocks with 4 spaces. Never use tabs.

Implied line continuation should align wrapped elements vertically, or use a hanging 4-space indent. Closing brackets can be placed at the end of the expression or on separate lines (indented same as the opening bracket line).

*[Google Python Style Guide](https://google.github.io/styleguide/pyguide.html)*
