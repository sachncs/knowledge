## Summary

Python uses docstrings to document code. Always use the three-double-quote format.

A docstring should be organized as a summary line (one physical line not exceeding 80 characters) terminated by a period, question mark, or exclamation point. When writing more, follow with a blank line and the rest of the docstring at the same cursor position as the first quote.

*[Google Python Style Guide](https://google.github.io/styleguide/pyguide.html)*
