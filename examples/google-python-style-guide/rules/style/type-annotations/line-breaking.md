## Summary

Try to follow the existing indentation rules when annotating.

After annotating, many function signatures will become one parameter per line. To ensure the return type also gets its own line, a comma can be placed after the last parameter.

If a single name and type is too long, consider using an alias for the type.

*[Google Python Style Guide](https://google.github.io/styleguide/pyguide.html)*
