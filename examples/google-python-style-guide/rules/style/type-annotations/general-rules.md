## Summary

Familiarize yourself with type hints. Annotating self or cls is generally not necessary.

- If any variable or returned type should not be expressed, use `Any`.
- You are not required to annotate all the functions in a module. At least annotate your public APIs.

*[Google Python Style Guide](https://google.github.io/styleguide/pyguide.html)*
