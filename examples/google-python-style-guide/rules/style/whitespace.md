## Summary

Follow standard typographic rules for the use of spaces around punctuation.

- No whitespace inside parentheses, brackets or braces
- No whitespace before a comma, semicolon, or colon
- Do use whitespace after a comma, semicolon, or colon (except at end of line)
- No whitespace before the open paren/bracket for argument lists, indexing or slicing
- No trailing whitespace
- Surround binary operators with a single space on either side
- Never use spaces around `=` when passing keyword arguments or defining default parameter values (exception: when a type annotation is present)
- Don't use spaces to vertically align tokens on consecutive lines

*[Google Python Style Guide](https://google.github.io/styleguide/pyguide.html)*
